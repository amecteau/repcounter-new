## Goal

Allow users to choose between Light, Dark, and "Match system" color themes. The preference is persisted in `UserSettings` alongside the existing language preference. "Match system" reads `prefers-color-scheme` at launch. The Settings screen hosts the control (similar in shape to `LanguageControl`).
